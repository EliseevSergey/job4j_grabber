Проект "Агрегатор Java-вакансий с Хабра".  

Программа считывает все вакансии с первых пяти страниц https://career.habr.com относящихся к Java и записывает их в базу данных pSql.  
Система запускается по расписанию - раз в минуту.  
Период запуска quartz можно указать в настройках -app.properties.

Стек: Java, JDBC, pSql, quartz.
